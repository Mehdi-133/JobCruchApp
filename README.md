# Job Dating

A modern PHP web application for managing job dating events and recruitment sessions.

## About

Job Dating is a platform designed to facilitate speed recruiting events where employers and job seekers can connect efficiently through structured interview sessions.

## Features

- User authentication and authorization
- Session management
- MVC architecture
- Secure routing system
- Database integration
- Input validation

## Requirements

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Composer
- Apache/Nginx web server
